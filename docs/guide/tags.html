4 Tags reference

The plugin defines the following custom tags:

TagDescription
resourcesCreates the necessary import for the CKEditor's javascripts files
configConfigure the editor instance
editorRenders the wysiwyg editor
fileBrowserCreates a link to open just the file browser
fileBrowserLinkCreates an URL to a file browser

all the tags belongs to the namespace ckeditor
